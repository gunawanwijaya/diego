# diego
diego is a business suite solution for SME, providing interface for business owner to interact with employees,
customers*, supply-chain providers and all actors involved in the business process.

<sup>* only supported in cloud tier</sup>

## Features
- open-source air-gapped solution for business like store and restaurant with features:
    1. basic financial report system
    1. basic ordering system
    1. inventory system
    1. transaction system
    1. API extension via gRPC (secure tls only)
- options to upgrade to cloud tier with benefits of:
    1. advance financial report system
    1. advance ordering system
    1. credit based payment system
    1. customer loyalty & membership system
    1. employee management system
    1. vouchers management system
